# Prompt Optimizer\n\nA comprehensive prompt optimization platform offering both MCP (Model Context Protocol) integration and a REST API service for improving AI prompts across various optimization goals.\n\n## 🚀 Components\n\nThis repository contains documentation and resources for:\n\n1. **[MCP Server Package](#mcp-server-package)** - NPM package for Claude Desktop, Cursor, Windsurf integration\n2. **[REST API Service](#rest-api-service)** - FastAPI backend for direct integration\n3. **[Web Dashboard](#web-dashboard)** - Browser-based interface and management\n\n---\n\n## 📦 MCP Server Package\n\n**Published on NPM**: [`mcp-prompt-optimizer`](https://www.npmjs.com/package/mcp-prompt-optimizer)\n\n### Quick Start\n\n```bash\n# Install globally\nnpm install -g mcp-prompt-optimizer\n\n# Setup your API key\nmcp-prompt-optimizer --setup\n```\n\n### Features\n\n- 🤖 **Universal MCP Support** - Works with Claude Desktop, Cursor, Windsurf, and any MCP-compatible client\n- 🔐 **Secure API Key Authentication** - Uses your personal API key for secure access\n- 🎯 **Advanced Optimization Goals** - Support for clarity, conciseness, technical accuracy, and more\n- 📊 **Real-time Quota Tracking** - See your usage and remaining quota\n- 💾 **Automatic Template Saving** - All optimizations saved with rich metadata for future reference\n- 🚀 **Easy Setup** - Simple configuration process\n- ⚡ **Production Backend** - Powered by enterprise-grade FastAPI deployment\n\n### MCP Client Configuration\n\n#### Claude Desktop\n\nAdd to `~/.claude/claude_desktop_config.json`:\n\n```json\n{\n  \"mcpServers\": {\n    \"prompt-optimizer\": {\n      \"command\": \"npx\",\n      \"args\": [\"mcp-prompt-optimizer\"]\n    }\n  }\n}\n```\n\n#### Cursor\n\nAdd to `~/.cursor/mcp.json`:\n\n```json\n{\n  \"mcpServers\": {\n    \"prompt-optimizer\": {\n      \"command\": \"npx\",\n      \"args\": [\"mcp-prompt-optimizer\"]\n    }\n  }\n}\n```\n\n#### Windsurf\n\nAdd via Windsurf settings:\n\n```json\n{\n  \"mcpServers\": {\n    \"prompt-optimizer\": {\n      \"command\": \"npx\",\n      \"args\": [\"mcp-prompt-optimizer\"]\n    }\n  }\n}\n```\n\n### Usage Example\n\nOnce configured, use the `optimize_prompt` tool in your MCP client:\n\n```\nPlease optimize this prompt using the optimize_prompt tool:\n\"Write me some code for a login system\"\n\nUse goals: [\"clarity\", \"specificity\", \"technical_accuracy\"]\n```\n\nResult:\n```\nCreate a secure user authentication system with the following specifications:\n\n## Requirements:\n- Implement login functionality with username/email and password\n- Include proper input validation and sanitization  \n- Add password hashing using bcrypt or similar\n- Implement session management\n- Include error handling for invalid credentials\n- Add rate limiting to prevent brute force attacks\n\n## Deliverables:\n- Backend authentication logic\n- Frontend login form\n- Database schema for user accounts\n- Security best practices implementation\n\nConfidence Score: 0.87\nQuota Remaining: 4,850/5,000\n```\n\n---\n\n## 🔗 REST API Service\n\n**Production Endpoint**: `https://p01--project-optimizer--fvrdk8m9k9j.code.run`\n\n### API Authentication\n\nAll API requests require an API key in the header:\n\n```bash\nX-API-Key: sk-opt-your-api-key-here\n```\n\n### Core Endpoint\n\n**POST** `/api/v1/optimize`\n\n```bash\ncurl -X POST \"https://p01--project-optimizer--fvrdk8m9k9j.code.run/api/v1/optimize\" \\\n  -H \"Content-Type: application/json\" \\\n  -H \"X-API-Key: sk-opt-your-api-key\" \\\n  -d '{\n    \"prompt\": \"Write me some code\",\n    \"goals\": [\"clarity\", \"specificity\"]\n  }'\n```\n\n### Response Format\n\n```json\n{\n  \"optimized_prompt\": \"Create a well-documented code solution with the following specifications...\",\n  \"confidence_score\": 0.87,\n  \"optimization_applied\": [\"clarity\", \"specificity\"],\n  \"quota_remaining\": 4,850,\n  \"template_saved\": true,\n  \"request_id\": \"req_abc123\"\n}\n```\n\n### Available Endpoints\n\n- `GET /health` - Service health check\n- `POST /api/v1/validate-key` - Validate API key\n- `POST /api/v1/optimize` - Optimize prompt\n- `GET /api/v1/quota` - Check usage quota\n- `GET /api/v1/templates` - List saved templates\n\n---\n\n## 🎯 Optimization Goals\n\nChoose
